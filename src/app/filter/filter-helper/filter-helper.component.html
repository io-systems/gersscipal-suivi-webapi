Liste des champs disponibles
  • status : état des messages
  • ofnr : numéro d'ordre de fabrication
  • codem : code du poste de travail
  • alea : Code aléa
  • operation : code opération
  • horodatages : voir pavé horodatages
 Horodatages

Pour filtrer les données sur un jour particulier :

  • jour: 24/04/2021
  • jour: today
  • jour: hier

Pour filtrer les données sur une période :

  • période: 24/04/2021 08:00:00, 26/04/2021 12:00:00
 Langage du filtre

Liste des caractères spéciaux :

  • : (deux points) = "CONTIENT", succède toujours à un champ
  • , (virgule) = "OU", est toujours placé entre deux valeurs
  • ; (point-virgule) = "ET", précède toujours un champ
 Exemple

Le filtre :

  • "alea: 02, 04; jour: hier"

se traduit par :

  • "Filtrer les données dont le champ Code aléa CONTIENT "02" OU "04" ET dont le champ horodatage API CONTIENT "la date d'hier""
